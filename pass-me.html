Pass Me

Overview

Remember it. Repeat it. Pass it on.

Pass Me is a fun multiplayer memory challenge designed for family and friends.

Memorise sequences, repeat them correctly, and pass the device to the next player. As rounds progress, patterns become increasingly difficult until only one memory champion remains.

Play solo or compete with up to four players across categories including colours, animals, faces, and objects.

Features

  • Single-player memory training
  • Multiplayer mode for up to four players
  • Pattern recognition gameplay
  • Categories including colours, faces, objects and animals
  • Quick rounds for family and friends
  • Easy to learn and fun to master
